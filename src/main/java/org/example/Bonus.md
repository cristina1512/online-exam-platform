Functionalitate prin care studentii pot depune contestatie
impotriva a cum a fost corectat raspunsul lor

Clasa noua: Contestatie 
Poate contine atribute precum: motivul contestatiei, daca contestatia a fost evaluata sau nu,
scorul inainte de contestatie, scorul dupa contestatie, data in care a fost depusa
contestatia (putem avea si o limita pana cand se pot depunde), data in care a fost finalizata
evaluarea contestatiei, o descriere scrisa de profesorul ce a evaluat contestatia
